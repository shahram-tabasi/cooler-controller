## ⚙️ Technical Overview

This project is developed for the **ATmega64A** microcontroller  
and runs with a **16 MHz external crystal oscillator**.

- Architecture: AVR 8-bit
- Application Type: Embedded System
- No external RAM used
- Stack size configured to 1024 bytes

## 🚀 Getting Started

### 🔧 Requirements

- AVR-GCC / CodeVisionAVR
- AVR Programmer (e.g., USBasp)
- ATmega64A Microcontroller
- 16 MHz Crystal Oscillator

### 🔨 Build & Flash

1. Compile the source code.
2. Generate the HEX file.
3. Flash it to the ATmega64A.
4. Power up and test the system.

---

## 📝 Notes

- Make sure fuse bits are configured for external 16 MHz crystal.
- Monitor stack usage if expanding the project.
- No external RAM is configured.
